# Hermes 任务编排场景存放目录

此目录用于存放 Hermes 任务编排的场景配置文件。

## 场景文件格式

每个场景应包含：
- 场景名称和描述
- 任务流程定义
- 参数配置
- 成功/失败处理规则

## 示例场景

### 1. 机房温度监测场景
- 定期获取机房温度数据
- 超过阈值时触发告警
- 通知飞书群组

### 2. 电源管理场景
- 监测 UPS 电池电量
- 预测供电时间
- 当电量不足时启动应急预案

### 3. 网络设备检测场景
- 自动检测网络设备状态
- 识别故障端口
- 生成运维报告

## 添加新场景

1. 在此目录创建场景文件（JSON/YAML 格式）
2. 在后端 `config/scenarios.py` 中注册场景
3. 通过 API 端点查询和执行场景
