# Hermes · 知识库

> Token 母体 · 机房智能运维 AI Agent 的全部知识 + 技能

## 快速开始
- 打开 `/console` → 「Agent 运营管理平台」→ 顶部「🧠 Hermes 知识库」
- 树状浏览：soul.md → domains/ → skills/
- 点击 skill → 查看 SKILL.md + py 代码 + 版本历史
- 点击「迭代」→ LLM 升级当前 Skill

## 目录索引
| 路径 | 说明 |
|------|------|
| `soul.md` | 全局灵魂 |
| `changelog.md` | 变更日志 |
| `domains/{id}/` | 6 大专业（env/fire/cctv/security/openclaw/ticket） |
| `domains/{id}/skills/{name}/` | 单个 Skill（SKILL.md + v*.py + archive/ + eval/） |
| `shared/` | 跨域共享工具 |
| `research/` | 项目自带资料（notes + reports） |
| `.hermes/diff/` | 升级 diff 历史 |
