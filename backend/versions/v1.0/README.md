# Console v1.0 备份说明

**备份时间**：2026-06-27
**内容**：系统总览（含 6 系统指标 / 3D 楼栋 / 实时告警 / 系统与人工双栏 / 子系统细节 / AI 浮窗）

## 备份文件
- `console.html` - 控制台前端页面
- `main.py` - 后端入口

## v1.0 主要特性
1. 白色 + 浅蓝色主题
2. 上方 6 大子系统核心统计仪表盘
3. 中部 3D 楼栋视图 + 实时告警
4. 下方 系统自动化 与 人工协作 双栏
5. 左侧菜单切换子系统细节数据
6. 右下角 AI Agent 浮窗，多上下文

## v1.0 → v2.0 主要变化
1. 新增 `Agent 运营管理平台` 页面（与系统总览同级）
2. 新增 Hermes 知识图谱、技能迭代、系统配置三大模块
3. 中央机器人 + 左右双区：报警日志、AI 自处理、AI 派发工单
