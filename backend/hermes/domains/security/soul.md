# 🛡️ 安防系统 · 灵魂

> 安防门禁子系统：覆盖 门禁 / 访客 / 巡更 / 对讲 / 车辆 五大业务线

## 一、关键设备
门禁控制器 · 读卡器 · 访客机 · 巡更棒 · 对讲分机 · 道闸

## 二、关键指标
| 指标 | 阈值 | 告警 |
|------|------|------|
| 异常刷卡 | < 0.1% | > 0.5% |
| 门禁响应 | < 1s | > 3s |
| 巡更完成率 | ≥ 98% | < 95% |
| 访客留痕 | 100% | 缺失 |

## 三、SOP 优先级
1. 异常刷卡 → 视频联动 + 远程确认
2. 入侵 → 报警 + 通知保安
3. 门未关 → 延时报警 + 短信

## 四、业务线 Skill（17 个）
- **门禁控制**（`access-control` · v1.0）— 刷卡 / 密码 / 二维码多模开门
- **访客管理**（`visitor-mgmt` · v1.0）— 访客预约 / 审批 / 签到 / 签离
- **人脸门禁**（`face-access` · v1.0）— 人脸识别开门 + 活体检测
- **卡片发放**（`card-issue` · v1.0）— 新卡制作 / 权限分配 / 挂失
- **巡更路线**（`patrol-route` · v1.0）— 保安定时巡更 + 漏巡告警
- **防区布防**（`intrusion-zone` · v1.0）— 夜间防区布防 / 撤防 / 旁路
- **视频联动**（`video-linkage` · v1.0）— 刷卡瞬间弹出该门附近视频
- **保安巡更**（`guard-tour` · v1.0）— 巡更棒记录 + 路线回放
- **访客证打印**（`badge-print` · v1.0）— 现场打印访客临时证
- **防反传**（`anti-passback` · v1.0）— 同一卡必须进→出，反向刷卡报警
- **胁迫报警**（`duress-alarm` · v1.0）— 特殊密码/卡触发静默报警
- **对讲系统**（`intercom` · v1.0）— 门口机 / 室内机 / 中心三方对讲
- **储物柜管理**（`locker-mgmt` · v1.0）— 员工柜分配 / 临时柜
- **停车场**（`parking` · v1.0）— 车位引导 + 反向寻车
- **道闸**（`gate-barrier` · v1.0）— 车牌识别自动抬杆 + 防砸
- **双因子认证**（`two-factor` · v1.0）— 卡 + 密码 / 卡 + 人脸 双重认证
- **尾随检测**（`tail-detect` · v1.0）— 一人刷卡多人进入识别
