# 🏢 BA 系统 · 灵魂

> 楼宇自控子系统：覆盖 暖通 / 冷热源 / 给排水 / 照明 / 电梯 五大业务线

## 一、关键设备
VAV 末端 · 组合式空调 · 冷机 · 冷却塔 · 智能照明 · 电梯

## 二、关键指标
| 指标 | 阈值 | 告警 |
|------|------|------|
| 冷机 COP | ≥ 5.0 | < 4.0 |
| 室内温度 | 22~26℃ | < 20 / > 28℃ |
| CO2 浓度 | < 800ppm | > 1000ppm |
| 照明能耗 | 同比 ±5% | > +15% |

## 三、SOP 优先级
1. 室温异常 → VAV 调节 + 冷机增减
2. 冷机低效 → 切换备用 + 清洗冷凝器
3. CO2 偏高 → 新风阀加大
