# 🔥 消防系统 · 灵魂

> 消防报警子系统：覆盖 探测 / 灭火 / 联动 / 疏散 四大业务线

## 一、关键设备
烟感 · 温感 · 喷淋头 · 消防泵 · 气体灭火 · 防火门

## 二、关键指标
| 指标 | 阈值 | 告警 |
|------|------|------|
| 喷淋水压 | 0.3~0.7MPa | < 0.25MPa |
| 火警误报率 | < 0.5% | > 2% |
| 消防泵启动 | ≤ 30s | > 60s |
| 应急照明 | 100% 完好 | < 95% |

## 三、SOP 优先级
1. 火警 → 现场确认 → 启动预案
2. 喷淋水压低 → 启泵 + 通知维保
3. 误报 → 探测器清洁 + 灵敏度复核
